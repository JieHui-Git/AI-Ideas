A simple poll creator where you can make a poll and see vote counts. The frontend is built using React + Vite, and the backend uses Express to handle API requests with JSON file storage.
==SETUP==
1. cd backend && npm install
2. cd frontend && npm install
3. Run backend: node server.js
4. Run frontend: npm run dev

*Note:* The backend stores data in a `polls.json` file located inside the `backend/` folder. Ensure this file exists before starting the application.
