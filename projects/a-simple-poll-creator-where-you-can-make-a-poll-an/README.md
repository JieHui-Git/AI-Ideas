# A simple poll creator where you can make a poll and see vote counts

Generated by [Vibe Coder](https://github.com/JieHui-Git/vibe-coder-application).

## Stack

- Frontend: React + Vite
- Backend: Express + JSON file storage

## Setup

1. cd backend && npm install
2. cd frontend && npm install
3. Run backend: node server.js
4. Run frontend: npm run dev
