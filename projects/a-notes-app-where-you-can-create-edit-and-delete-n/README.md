# a notes app where you can create edit and delete notes

Generated by Vibe Coder.

## Stack

- Frontend: React + Vite + Express
- Backend: Express + SQLite

## Setup

1. cd backend && npm install express sqlite3
2. cd frontend && npm install vite @vitejs/plugin-react react-router-dom uuid
3. Run backend: node server.js
4. Run frontend: npm run dev
