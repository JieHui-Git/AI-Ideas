# a notes app where you can create edit and delete notes

Generated by Vibe Coder.

## Stack

- Frontend: React + Vite + Express + SQLite
- Backend: Express

## Setup

1. cd backend && npm install express sqlite3 body-parser cors
2. cd frontend && npm install vite @vitejs/plugin-react react-router-dom
3. Run backend: node main.js
4. Run frontend: npm run dev
